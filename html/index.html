About

skill

  • HTML

    웹 표준성을 준수하는 HTML 구조 설계를 통해 퍼블리싱을 해오고 있습니다. HTML에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다. HTML의 모든 태그 사용법을 알고 있는 것은 아니지만 웹사이트 제작시 자주 사용하는 태그와 이에 대한 속성은 반복적으로 접하다보니 이해하면서 사용가능한 수준입니다.

    HTML5

    80%

  • CSS

    색상의 효과와 크기조절을 통해 깔끔한 디자인을 만들기 위해 노력하고 있습니다. flex를 통한 배치를 먼저 작업하는 것을 선호하며 position속성과 애니메이션 효과를 통해 다양한 디자인을 구축하고 있습니다. 반응형 작업 시 float, width, height, flex를 활용해 전체 사이즈를 재단하며 그 외에 세부적인것을 수정해 미디어쿼리 작업을 할 수 있습니다.
    SCSS는 설치 및 save path활용, 부모, root를 활용할 수 있는 수준이며 따로 공부를 하면서 배우고 있습니다.

    CSS

    70%

  • JavaScript

    Javascript의 실력은 다른 것들에 비해 부족한 상태입니다. 제가 할 수 있는 능력은 Javascript의 기본 형식이나 연산자를 이해하는 수준입니다. 또한, if문, for문, switch구문의 사용 방법과 해석정도 가능한 상태입니다. jQuery가 가독성이나 코드 짜는데 편하다는 장점이 있지만 속도 차이에 있어 Javascript가 빠르다는 것을 알게 되었습니다. 사용자의 편의를 위해 Javascript를 배우면서 익숙해지고 숙달되기 위해 노력하겠습니다.

    Javascript
